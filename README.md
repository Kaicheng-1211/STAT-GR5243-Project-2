# STAT-GR5243-Project-2: Web Application Development and Deployment using R Shiny



## Project Overview

This project covers the full data science workflow. The goal is to derive insights on urban transportation patterns from **NYC TLC** trip records. Deliverables include a reproducible workflow: web-based data acquisition, data cleaning, feature engineering, and trend visualization.

## Requirements

- **R** (version 4.0 or higher)
- **RStudio** (recommended)
- Dependencies listed in `requirements.txt`
